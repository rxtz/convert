# convert

## `angle`

### Usage

```bash
$ angle
Angle: <>
Sexagesimal/Centesimal/Radian [s/c/r]: <>
```

```bash
$ angle <> [s/c/r]
```

### Example

```bash
$ angle 180 s
180.00°
200.00ᵍ
1.00π rad
```

## `temperature`

### Usage

```bash
$ temperature
Temperature: <>
Celsius/Fahrenheit/Kelvin [c/f/k]: <>
```

### Example

```bash
$ temperature
Temperature: 16
Celsius/Fahrenheit/Kelvin [c/f/k]: c
16 °C
60.8 °F
289.15 K
```
